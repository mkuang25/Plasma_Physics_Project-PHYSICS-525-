# Plasma_Physics_Project-PHYSICS-525
This code plots dispersion relations for a 2D coupled and uncoupled plasma sheets with and without temperature dependence, in addition to their group velocities to investigate how these oscillations propogate in 2D plasmas. 

# EDITING PARAMETERS
The code simply plots analytically derived relations. To edit the parameters, simply change the corresponding parameter in the section of the code titled "Constants and Parameters".
